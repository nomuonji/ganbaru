# 📺 YouTubeチャンネル プロフィール

## チャンネル基本情報

### チャンネル名
**今日の頑張り**
（英語表記: Today's Ganbaru）

### ハンドル
`@kyono-ganbari` または `@todays-ganbaru`

### チャンネル説明文
```
🌅 朝、今日の目標をコメントで宣言！
🌙 夜、今日できたことをコメントで報告！
✨ みんなの頑張りを毎日動画でシェアします

━━━━━━━━━━━━━━━━━━━━

【このチャンネルについて】
「今日の頑張り」は、みんなで毎日の小さな目標を共有し、
一緒に頑張るコミュニティチャンネルです。

📌 毎日3本の動画を投稿：
・朝7時 → 「今日の目標は？」動画
・夜9時 → 「今日できたことは？」動画
・夜11時半 → みんなの頑張りまとめ動画

💡 参加方法：
1. 朝の動画にあなたの今日の目標をコメント
2. 夜の動画に今日できたことをコメント
3. まとめ動画であなたのコメントがアニメーションで紹介されます！

小さな一歩も、みんなで共有すれば大きな力に。
一人じゃ続かないことも、仲間がいれば頑張れる。

明日も一緒に頑張ろう！💪

━━━━━━━━━━━━━━━━━━━━

#今日の頑張り #モチベーション #毎日投稿 #目標達成 #習慣化
```

## コンテンツ戦略

### 動画カテゴリ
1. **朝ショート** (Shorts)
   - 投稿時間: 毎日 7:00
   - 形式: 縦長 (9:16)
   - 長さ: 15秒
   - 内容: 今日の目標を募集

2. **夜ショート** (Shorts)
   - 投稿時間: 毎日 21:00
   - 形式: 縦長 (9:16)
   - 長さ: 15秒
   - 内容: 今日できたことを募集

3. **まとめ動画**
   - 投稿時間: 毎日 23:30
   - 形式: 横長 (16:9)
   - 長さ: 1〜3分（参加者数による）
   - 内容: コメントをアニメーションで紹介

### ハッシュタグ戦略
