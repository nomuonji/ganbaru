# 🎨 プロフィール画像・バナー デザイン仕様書

## プロフィール画像（チャンネルアイコン）

### 基本情報
- **サイズ**: 800 x 800 px（推奨）
- **形式**: PNG（透過なし）
- **表示サイズ**: 98 x 98 px（縮小表示されるため、シンプルに）

### デザインコンセプト
**「朝と夜、一日を通して頑張る」を象徴する太陽と月の融合**

### レイヤー構成

```
レイヤー1: 背景（円形グラデーション）
├── 左側: #FF6B6B → #FF8E53（朝焼けオレンジ）
├── 中央: #764ba2（パープル）
└── 右側: #1a1a2e → #0f3460（夜空ネイビー）

レイヤー2: 太陽（左側）
├── 位置: 円の左側 40%
├── 色: #FFD700（ゴールド）
├── 効果: 放射状の光線（5-7本）
└── サイズ: 直径の35%

レイヤー3: 月（右側）
├── 位置: 円の右側 40%
├── 色: #f5f5dc → #fffacd（クリーム〜レモン）
├── 形状: 三日月
└── サイズ: 直径の30%

レイヤー4: 装飾
├── 星: 3-5個の小さな星（#FFFFFF、輝き効果）
├── キラキラ: 中央に✨エフェクト
└── 位置: 太陽と月の間

レイヤー5: 縁取り（オプション）
└── 3px の白い縁取り
```

### カラーパレット
```css
--sunrise-start: #FF6B6B;
--sunrise-end: #FFC857;
--sunset-purple: #764ba2;
--night-start: #1a1a2e;
--night-end: #0f3460;
--sun-gold: #FFD700;
--moon-cream: #f5f5dc;
--star-white: #FFFFFF;
```

### Figma/Canvaでの作成手順

1. **新規キャンバス**: 800 x 800 px
2. **背景円を作成**: 
   - 円形を描く（780 x 780 px、中央配置）
   - 線形グラデーション（左: #FF6B6B → 中央: #764ba2 → 右: #0f3460）
3. **太陽を追加**:
   - 円形（直径 280 px）
   - 位置: 左から 160 px
   - 色: #FFD700
   - 影: 内側グローで立体感
   - 光線: 細い三角形を放射状に配置
4. **月を追加**:
   - 三日月シェイプ
   - 位置: 右から 160 px
   - 色: #f5f5dc
   - 影: 外側グロー（#fffacd, 20px）
5. **星を追加**:
   - 4点/6点スターシェイプ
   - 3-5個をランダムに配置
   - サイズ: 15-30 px
   - 色: #FFFFFF
   - 効果: 外側グロー
6. **仕上げ**:
   - 全体に軽いノイズテクスチャ（5%）
   - 白い縁取り（3px）

- **方向**: 左から右

| 要素 | フォント | サイズ | 色 | スタイル |
|------|---------|--------|-----|---------|
| チャンネル名 | Noto Sans JP | 120px | #FFFFFF | Bold, 影あり |
| サブタイトル | Noto Sans JP | 48px | rgba(255,255,255,0.9) | Medium |
